导入/导出说明

导出功能:

  • JSON格式: 包含完整数据，适合备份和迁移
  • CSV格式: 适合在电子表格中打开和分析

导入功能:

  • 支持多个文件同时导入
  • 系统会自动去重（基于时间戳和节点）
  • 导入数据会与现有数据合并
  • 导入后请运行数据分析以更新统计信息

数据格式:

{
  "timestamp": "2024-01-01T12:00:00Z",
  "userLocation": "shenzhen",
  "mainlandNode": "cn_bj_cloud",
  "mainlandLatency": 45,
  "taiwanNode": "tw_nchc",
  "taiwanLatency": 68,
  "period": "peak"
}
多设备数据共享

选择共享组

选择相同的共享组，即可在不同设备间自动同步数据

提示：在所有设备上选择相同的共享组，数据将自动同步

使用方法

1. 选择共享组（如 Group_001）

2. 点击"保存共享组"按钮

3. 在其他设备上重复步骤1-2

4. 使用"同步到云端"和"从云端同步"按钮同步数据

5. 数据将自动在所有设备间共享
