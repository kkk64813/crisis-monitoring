导入/导出说明

导出功能:

  • JSON格式: 包含完整数据，适合备份和迁移
  • CSV格式: 适合在电子表格中打开和分析

导入功能:

  • 支持多个文件同时导入
  • 系统会自动去重（基于时间戳和节点）
  • 导入数据会与现有数据合并
  • 导入后请运行数据分析以更新统计信息

数据格式:

{
  "timestamp": "2024-01-01T12:00:00Z",
  "userLocation": "shenzhen",
  "mainlandNode": "cn_bj_cloud",
  "mainlandLatency": 45,
  "taiwanNode": "tw_nchc",
  "taiwanLatency": 68,
  "period": "peak"
}

Cloudflare 使用说明

什么是 Cloudflare Workers？

  • Cloudflare Workers 是 Cloudflare 提供的无服务器计算平台
  • 免费套餐提供每天 100,000 次请求
  • 数据存储在 Cloudflare 的全球边缘网络中
  • 支持跨设备访问同一份数据

同步功能说明:

  • 同步到云端: 将本地配置和历史数据上传到 Cloudflare
  • 从云端同步: 从 Cloudflare 下载数据到本地
  • 完整备份: 创建包含所有数据的备份文件
  • 恢复备份: 从备份文件恢复数据

注意事项:

  • 数据在 Cloudflare 中是加密存储的
  • 每个用户ID对应一个独立的数据空间
  • 免费套餐有每日请求限制，正常使用完全足够
  • 建议定期导出本地备份以防万一
